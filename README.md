# Bank Campaign Analytics

End-to-end data analysis project on UCI Bank Marketing Dataset.

## What's included
- SQL EDA (7 phases, 30+ queries)
- Python/pandas EDA with matplotlib/seaborn
- Statistical validation (chi-square, t-test, point-biserial)
- Power BI dashboard (4 pages, interactive)

## Key findings
- Call duration is the strongest predictor (r=0.45)
- Campaign fatigue starts after 3 calls
- Students and retirees convert at 74.7% and 66.3%
- Re-contact window of 31-90 days converts at 81.7%
